# 🏥 パッとシフト管理システム

看護師向けのシフト管理アプリケーションです。

有給休暇の取得状況や夏季休暇の管理、勤務ルールの確認など、シフト作成時の負担軽減を目的として制作しました。

## 開発背景

シフト作成では、

- 有休取得状況の管理
- 夏季休暇の管理
- 勤務ルールの確認
- 月ごとのカレンダー作成

など、多くの確認作業が発生します。

これらを一つの画面で管理し、入力ミスや確認漏れを減らすことを目的として制作しました。

---

## 主な機能

### スタッフ画面

- 公休・有休・夏休の申請
- 有休取得状況の表示
- 申請上限のリアルタイムチェック

### 管理者画面

- 年・月を選択したカレンダー自動生成
- 勤務ルールのON/OFF切り替え
- ルール違反のリアルタイム表示
- Gemini APIを利用したシフト分析（試作）

---

## 工夫した点

### カレンダーを動的に生成

選択した年・月から曜日と日数を自動計算し、毎月コードを書き換えなくても利用できるようにしました。

### 勤務ルールを自動判定

「夜勤明け翌日の日勤」などの勤務パターンを検知し、リアルタイムで警告を表示する仕組みを実装しました。

### 入力ミスを防止

有休残日数や夏休の上限を超える申請をブラウザ上で判定し、誤入力を防止しています。
